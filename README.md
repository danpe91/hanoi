Este proyecto cuenta con el famoso juego de torres de hanoi.

El juego consiste en mover las fichas de la torre origen a la 
torre destino, para esto contamos con una torre auxiliar en la 
que podemos ir acomodando los discos para ayudarnos a cumplir el objetivo.

Este proyecto incluye un módulo que soluciona el problema automáticamente,
el cual realiza la solución de 4,5,6, ó 7 discos, eligiendo esto aleatoriamente.

Incluye tambien un módulo en el cual el usuario puede jugar libremente
para cumplir el objetivo. Este modo guarda las puntuaciones del usuario, basandose
en el número de movimientos y el tiempo que se emplea para solucionar.

Y por último tambien incluye un módulo en el que podremos consultar las puntuaciones
y ver las mejores puntuaciones de manera ordenada en base al número de movimientos 
o al tiempo empleado. Se puede seleccionar el número de discos para consultar las 
mejores por número de discos.

	Instrucciones

Para mover un disco de una torre a otra hay que dar clic en la torre origen para seleccionar
el disco a mover, al dar clic en esa torre automáticamente seleccionará la ficha superior
para mover ésta misma. Ya despues de haber dado clic a la torre origen de debe de dar clic en 
la torre destino, ya hecho esto se moverá el disco de la torre de origen a la torre seleccionada.

La selección no es demasiado precisa, para seleccionar la torre se puede dar clic en el
área que comprende la torre a la que damos clic.
